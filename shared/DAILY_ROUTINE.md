# Daily Routine

This is the schedule every restaurant agent follows.

## Schedule (in restaurant's local timezone)

### 8:00 AM — Morning Brief
- Pull yesterday's sales data from POS
- Calculate total sales, average ticket, top items
- Compare to same day last week
- Send morning summary to owner

### 12:00 PM — Lunch Check
- Check if lunch rush data is available
- If notable: send quick update
- If normal: skip (don't spam)

### 6:00 PM — Evening Report
- Compile full day's performance
- Identify trends
- Prepare actionable suggestions
- Send evening summary to owner

### 9:00 PM — End of Day
- Final sales numbers
- Weekly trend update
- Flag any issues (low sales, high costs)
- Prepare tomorrow's focus area

## Rules
- Don't send more than 3 messages per day unless owner asks
- Skip routine messages if no data available
- Always include context ("vs last week")
- One suggestion per message maximum
